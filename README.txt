Jogo do NIM

O jogo envolve a definição inicial de quantas peças estarão disponíveis no tabuleiro e o número máximo de peças que podem ser retiradas em cada turno. Os turnos alternam entre o computador e o usuário, sendo que em cada turno é obrigatório retirar pelo menos uma peça, até o limite máximo estabelecido. O objetivo é ser o jogador que retira a última peça, deixando o tabuleiro vazio, o que determina a vitória. É possível optar por uma partida única ou por uma série de três jogos para aumentar a competição. Boa sorte!
